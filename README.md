# Mimir

> Mimir is a figure from Norse mythology renowned for his wisdom and knowledge. After being beheaded during the Aesir-Vanir war, Odin preserves his head and consults it for advice and knowledge.

This project is a Rust implementation of Mini-Prolog, completed for a Computer Science dissertation. Project documentation can be found [here](https://kit-matthewson.github.io/mimir).
